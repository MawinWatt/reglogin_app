<h3>Installation</h3>

การสร้างเทมเพลต Next.js
```
npx create-next-app
```

Package Installation on both Frontend and Backend directories
```
npm install
```

Start server on both Frontend and Backend directories
```
npm run dev
```

เข้าสู่โปรเจค
```
http://localhost:3000/
```
